## Infrastructure Notes

A few things worth knowing before you buy:

**KVM vs OpenVZ**: The Ryzen VPS plans (including the $7/year deal) run proper KVM virtualization — true VM isolation, no shared kernel, Docker-friendly. They also offer older OpenVZ-based budget plans for lower-cost use cases, though the company has said they plan to migrate all OpenVZ customers to KVM over time.

**Chicago connectivity**: One location, good North American coverage, 1Gbps standard ports. If your audience is primarily in the US, this works well. If you need European or Asian edge, you'd want a different provider.

**DDoS protection**: Included across all plans. It's basic automatic filtering and null-routing — enough to handle typical volumetric attacks, not enterprise-grade mitigation.

**IP-KVM on dedicated**: This is a genuinely useful feature. It means out-of-band access at the BIOS/hardware level, so you can reinstall the OS, troubleshoot boot failures, or recover from a misconfiguration without needing a provider to physically touch the machine.
